# Sound Files for Lucky Draw

ระบบต้องการไฟล์เสียง 3 ไฟล์ในโฟลเดอร์นี้:

## ไฟล์ที่ต้องการ:

1. **draw.mp3** - เสียงเมื่อเริ่มจับรางวัล (เสียงกลอง/เสียงสุ่ม)
2. **winner.mp3** - เสียงเมื่อประกาศผู้ชนะ (เสียงแฟนแฟร์/เสียงชนะ)
3. **click.mp3** - เสียงเมื่อคลิกปุ่ม Sound (เสียงคลิกสั้นๆ)

## แหล่งเสียงแนะนำ:

### ฟรีและใช้ได้:

- **Freesound.org** - https://freesound.org
- **Pixabay Sounds** - https://pixabay.com/sound-effects
- **Mixkit** - https://mixkit.co/free-sound-effects
- **Zapsplat** - https://www.zapsplat.com

### คำค้นที่แนะนำ:

- **draw.mp3**: "drum roll", "spinning wheel", "shuffle", "lottery"
- **winner.mp3**: "fanfare", "victory", "celebration", "win", "applause"
- **click.mp3**: "button click", "ui click", "switch"

## วิธีเพิ่มไฟล์:

1. ดาวน์โหลดไฟล์เสียงที่ต้องการ
2. แปลงเป็นไฟล์ .mp3 (ถ้ายังไม่ใช่)
3. เปลี่ยนชื่อไฟล์ให้ตรงกับที่ระบุข้างต้น
4. วางไฟล์ในโฟลเดอร์นี้

## หมายเหตุ:

- ขนาดไฟล์ไม่ควรเกิน 500KB ต่อไฟล์
- ควรใช้ไฟล์ที่มีความยาวไม่เกิน 3-5 วินาที
- ตรวจสอบลิขสิทธิ์ก่อนใช้งาน
- สามารถปรับระดับเสียงในไฟล์ให้เหมาะสมก่อนนำมาใช้

## ตัวอย่างเสียงที่เหมาะสม:

### draw.mp3 (3-5 วินาที)

- เสียงกลองแบบ drum roll
- เสียงวงล้อหมุน
- เสียงการ์ดสับ

### winner.mp3 (2-4 วินาที)

- เสียงทรัมเป็ต fanfare สั้นๆ
- เสียงระฆัง + applause
- เสียง "Ta-da!"

### click.mp3 (< 0.5 วินาที)

- เสียงคลิกเบาๆ
- เสียงสวิตช์
- เสียง UI feedback สั้นๆ
